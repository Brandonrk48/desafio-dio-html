Falando sobre Tags

Basicamente, as Tags são o que dão formato aos arquivos HTML. Mas de que forma?

As tags servem para formatar os textos e a forma que eles serão distribuidos na página web.

Individualmente, as Tags tem nomes e sintaxes exclusivas que determinam as formas que elas devem ser abertas e fechadas.

Exemplos

  • <h1>"Titulo do seu interesse"</h1>
    • Essa Tag serve para escrever títulos.
    • Os tamanhos vão do h1 até o h6, onde o h1 é a maior fonte.
  • <p>"Parágrafo"</p>
    • Tag que tem como finalidade a escrita de textos.
    • A sua fonte é menor.
  • <title>"Título do seu interesse"</title>
    • Essa Tag é escrita a nível de head(cabeça).
    • A estrutura head fica "fora do nível body".
    • O título escrito dentro dessa Tag, fica exposto no retângulo da janela do site.
    • Esse título carrega primeiro que a estrutura do body do arquivo HTML.
  • <mark>"Texto qualquer"</mark>
    • Essa Tag tem como finalidade marcar o texto.
    • O texto é demarcado de forma semelhante a um marca texto.

Atributos das Tags

Basicamente, os atributos são propriedades de formatação que uma Tag pode ter.

Existem atributos específicos de cada Tag, e também, atributos mais generalizados.

Exemplos

  • <strong id = "titulo">"texto de interesse"</strong>
    • Este atributo, "id", serve para indicar elementos que futuramente poderam ser manipulados ou acessados.
    • Ele pode ser acessado pelo Javascript e pelo css.
  • <strong style = "titulo">"texto de interesse"</strong>
    • Esse atributo "style = elemento de interesse" é utilizado em css.
    • Serve para formatar o HTML.

Textos

Por influência de o HTML ter a finalidade de formatar hypertextos, as suas Tags trazem essa finaliade para todos os conteúdos adicionados.

O navegador, por padrão, após formatarmos o texto de acordo com o nosso interesse, ele próprio, já cria um espaçamento entre o conteúdo das Tags.

Posteriormente, essa formatação pode ser alterada de acordo com o seu interesse, pelo css.

Exemplos

  • <h1>"Titulo do seu interesse"</h1>
    • Essa Tag serve para escrever títulos.
    • Os tamanhos vão do h1 até o h6, onde o h1 é a maior fonte.
  • <sup>"informação qualquer"</sup>
    • Essa Tag tem como finalidade, elevar uma informação como se fosse uma exponenciação.
  • <u>"informação qualquer</u>
    • Essa Tag serve para por um underline na informação desejada.

Listas ordenadas e não ordenadas

Essas funcionalidades tem diversas finalidades, como por exemplo, a criação de menus, listagem de conteúdos, etc.

As listas ordenadas enumeram os seus itens, diferentemente das listas não ordenadas, que apenas destacam os seus intens com pontos.

Dentro dessas listas, tanto das ordenadas quando as que não são ordenadas, para elas atenderem as espectativas e fucionarem perfeitamente, se deve adicionar outra Tag dentro delas, a "li".

Exemplos

  • <ol>"incluso na Tag ol, vai a Tag li, e dentro das duas, o conteúdo de interesse"</ol>
    • Essa Tag, é a que cria listas ordenadas, em outras palavras, listas enumeradas.
  • <ul>"incluso na Tag ul, vai a Tag li, e dentro das duas, o conteúdo de interesse"</ul>
    • Essa Tag, diferentemente da apresentada anteriormente, cria listas não ordenadas, listas marcadas apenas por pontos, da mesma forma que estamos usando aqui para dar exemplos, etc.

Links

Um site é basicamente conectado com diversos links.

Estes links podem nos redirecionar para páginas externas e também, para outras páginas internas do nosso site.

Além de os links nos direcionar para páginas internas e externas, podemos também, crias marcações em partes do arquivo HTML que temos interesse, para que a navegabilidade em nosso site ou página, seja mais otimizado e rápido, acessando esses "links criados em determinadas partes da página".

Para "linkarmos" algo do nosso interesse em nosso site, usamos a Tag "a" que significa ancora.

Os "links criados internamente na página, podem nos direcionar para qualquer parte da mesma."

Exemplos

  • <a = href "link externo" "nomenclatura escolhida para o link"> </a>
    • Essa Tag serve para adicionar à página, links de outros sites ou adicionar links de páginas do próprio site.
